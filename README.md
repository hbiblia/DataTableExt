# DataTableExt
DataTableExt son nodos extras para DataTable de UnrealEngine

- AddDataTableRow
  - Agrega una nueva row a la tabla siempre que no exista.
- SetDataTableRow
  - Actualiza los datos de una tabla siempre que exista. 
- RemoveDataTableRow
  - Borramos una row de una tabla.
